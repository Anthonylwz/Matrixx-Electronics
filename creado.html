Sobre Mí

Mi filosofía de desarrollo se basa en tres pilares: rendimiento, mantenibilidad y experiencia de usuario. Creo firmemente que una buena arquitectura de software es la base de cualquier proyecto exitoso. Por eso, estructuro mis aplicaciones de forma modular y escalable, incluso sin depender de frameworks pesados.

Disfruto enfrentando desafíos técnicos y transformando ideas complejas en interfaces elegantes y funcionales. El proyecto Matrixx Electronics es un claro ejemplo de esta pasión, donde cada componente fue diseñado desde cero para ofrecer una experiencia de compra fluida y completa.

Proyecto Destacado

Matrixx Electronics - Tienda E-Commerce

Una interfaz de tienda en línea completamente funcional y renderizada en el lado del cliente, construida desde cero con JavaScript puro. Este proyecto demuestra la capacidad de crear aplicaciones complejas y ricas en características sin la necesidad de un framework, centrándose en una base de código sólida y patrones de diseño modernos.

Características Clave Implementadas:

  • Arquitectura Orientada a Objetos: App estructurada en clases de JS para cada componente (Sliders, Carrito, Modal).
  • Gestor de Estado Central: Una clase `Store` controla el estado de la aplicación (productos, carrito) y notifica a los componentes usando un patrón Publisher/Subscriber.
  • Carrito de Compras Persistente: Funcionalidad completa para añadir, eliminar y actualizar productos, con persistencia de datos en `localStorage`.
  • Búsqueda y Filtrado Dinámico: Búsqueda en tiempo real con sugerencias y Debouncing, junto con filtros por categoría y múltiples opciones de ordenamiento.
  • UI Interactiva y Atractiva: Múltiples carruseles de productos, un modal de vista rápida, notificaciones dinámicas y un diseño totalmente responsivo.
  • Enfoque en Accesibilidad: Uso extensivo de atributos ARIA para una navegación accesible y compatible con tecnologías de asistencia.
